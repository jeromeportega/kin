# Project rules for Claude

## No AI attribution in trace history

Do not add any AI authorship attribution to anything that lands in version-control history. This includes, but is not limited to:

- `Co-Authored-By: Claude ...` trailers in commit messages
- "🤖 Generated with Claude Code" lines in commit messages or PR descriptions
- "Generated by Claude" comments in code
- AI-involvement mentions in branch names, PR descriptions, release notes, or changelog entries

Author the work as the user. This applies to every commit, PR, branch, and code comment regardless of how it was produced.
